# 이음기프트 — 대학 홍보물 기획 정리

> **목표**: 이음기프트를 "모바일 쿠폰 서비스"가 아니라,
> **"대학의 지급·보상 업무를 가장 쉽게 만드는 운영 솔루션"**으로 포지셔닝하는 것.

---

## 1. 전체 기획 논리

### 1-1. 핵심 프레임

기존 접근이 약한 이유는 **"이음기프트가 뭘 할 수 있는가"**를 설명하기 때문입니다.
바꿔야 할 프레임은 **"이음기프트 없이 지금 얼마나 번거로운가"**입니다.

```
[AS-IS] 이음기프트는 모바일 쿠폰을 대량 발송할 수 있는 서비스입니다
[TO-BE] 설문 200명 경품, 아직 편의점 가서 사오시나요?
```

| 단계 | 기존 (문제) | 이음기프트 (해결) |
|------|------------|-----------------|
| 상품 선정 | 예산 맞추려 비교 쇼핑 | 예산 입력하면 자동 추천 |
| 구매 | 직접 구매·재고 관리 | 모바일 쿠폰, 재고 없음 |
| 배분 | 명단 정리·현장 배부 | 엑셀 업로드 → 일괄 발송 |
| 수령 확인 | 서명·수기 관리 | 자동 수신 확인 |
| 정산 | 영수증·품의서 수작업 | 정산서 자동 발행 |

> **한 문장 요약**: "선정부터 정산까지, 5단계가 2단계로 줄어듭니다."

### 1-2. 설득 순서 (모든 자료의 공통 흐름)

```
① 공감 (지금 이 일, 번거롭지 않으세요?)
② 해결 (이음기프트로 이렇게 바뀝니다)
③ 증거 (이미 OO대학교 외 N개교 이용 중)
④ 행동 (지금 상담 신청하세요)
```

## 2. 타깃별 메시지 분리

### 2-1. 교직원·행정조직용

**이들의 진짜 고민**:
- 지급 건마다 품의→구매→배분→정산 반복
- 예산 집행 증빙 처리가 번거로움
- 소액 다건(설문 사은품 등)일수록 행정 부담 큼
- 담당자가 바뀌어도 돌아가야 함

**포지셔닝**: `학교 지급 업무의 운영 파트너`

| 메시지 요소 | 내용 |
|------------|------|
| **헤드라인** | "지급 업무, 절반으로 줄이는 방법" |
| **서브카피** | 설문 사은품부터 복지 지급까지 — 선정·발송·정산을 한 번에 |
| **핵심 키워드** | 운영 효율 / 정산 간소화 / 예산 맞춤 / 전담 지원 / 대량 처리 |
| **신뢰 포인트** | 도입 대학 수, 연간 처리 건수, 정산서 샘플 |
| **CTA** | "운영 상담 신청 →" (전화/QR) |

### 2-2. 행사 담당자(학생회)용

**이들의 진짜 고민**:
- 축제·이벤트 경품을 뭘로 할지 매번 고민
- 예산이 빠듯한데 실물 경품은 비쌈
- 현장에서 나눠주는 게 혼잡함
- 행사 끝나면 남은 경품 처리도 귀찮음

**포지셔닝**: `행사 경품 운영을 가볍게 만드는 도구`

| 메시지 요소 | 내용 |
|------------|------|
| **헤드라인** | "경품 준비, 5분이면 끝납니다" |
| **서브카피** | 실물 경품 없이 — 예산에 맞게 골라서, 바로 보내세요 |
| **핵심 키워드** | 간편 운영 / 즉시 발송 / 예산 맞춤 / 재고 부담 제로 / 남는 경품 없음 |
| **신뢰 포인트** | 활용 사례(축제, OT, 설문 등), 가격대별 상품 예시 |
| **CTA** | "지금 견적 받기 →" (카카오톡/QR) |

### 2-3. 메시지 차이 요약

| 구분 | 교직원용 | 행사 담당자용 |
|------|---------|-------------|
| **톤** | 신뢰·효율 중심 (B2B) | 가볍고 실용적 (B2C에 가까운 B2B) |
| **첫 인상** | "업무가 줄어든다" | "준비가 쉬워진다" |
| **강조점** | 정산·관리·반복 업무 해소 | 속도·간편함·예산 절약 |
| **증거 방식** | 정산서 샘플, 도입 기관 리스트 | 활용 사례, 상품 가격표 |
| **CTA 느낌** | 공식 상담 | 가볍게 문의 |

## 3. 전단지 앞면·뒷면 메시지 설계

### 3-1. 교직원용 전단지

#### 앞면 (3초 안에 꽂혀야 함)

```
[상단 — 공감 헤드라인]
"설문 경품 200개, 아직 직접 사서 나눠주시나요?"

[중앙 — 해결 한 줄]
선정부터 정산까지, 이음기프트가 한 번에.

[하단 — 핵심 수치 3개 (아이콘+숫자)]
✓ 상품 선정 → 3분    ✓ 대량 발송 → 1초    ✓ 정산 → 자동
```

#### 뒷면 (관심 생긴 사람이 확인)

```
[영역 1 — 이런 업무에 활용하세요]
· 설문·만족도 조사 사은품
· 행사·세미나 참가 보상
· 학생 복지 지급 (장학, 근로 등)
· 교내 이벤트 경품

[영역 2 — 왜 이음기프트인가 (3가지)]
① 예산에 딱 맞추는 상품 구성
② 엑셀 한 번이면 수백 건 일괄 발송
③ 정산서 자동 발행 → 증빙 걱정 없음

[영역 3 — 도입 실적]
"OO대학교 외 N개교 이용 중"

[영역 4 — CTA]
1:1 전담 매니저가 세팅부터 정산까지 지원합니다
📞 000-0000-0000 | [QR코드]
```

### 3-2. 행사 담당자용 전단지

#### 앞면

```
[상단 — 공감 헤드라인]
"축제 경품, 아직 직접 포장하고 계세요?"

[중앙 — 해결 한 줄]
모바일로 바로 보내세요. 남는 경품도, 긴 줄도 없습니다.

[하단 — 핵심 수치 3개]
✓ 3,000원부터 구성 가능    ✓ 즉시 발송    ✓ 재고 부담 0
```

#### 뒷면

```
[영역 1 — 이런 행사에서 쓰세요]
· 축제 부스 이벤트 경품
· OT / MT 미니게임 상품
· SNS 이벤트 리워드
· 설문 참여 감사 선물

[영역 2 — 이렇게 간단합니다 (3 STEP)]
① 예산 알려주세요 → 맞춤 상품 추천
② 명단(번호) 전달 → 일괄 발송
③ 끝. 정산서도 자동.

[영역 3 — 가격 예시표]
| 예산 | 추천 상품 예시 |
|------|-------------|
| 3,000원 | 편의점 음료 |
| 5,000원 | 카페 아메리카노 |
| 10,000원 | 치킨 / 피자 |
| 30,000원+ | 백화점 상품권 |

[영역 4 — CTA]
카톡으로 바로 견적 받으세요
💬 [카카오 QR코드]
```

| # | 기존 (설명형) | 개선 (메리트형) |
|---|-------------|---------------|
| 1 | 다양한 모바일 쿠폰을 제공합니다 | **3,000원부터 30,000원까지, 예산에 딱 맞는 상품** |
| 2 | 대량 발송 기능을 지원합니다 | **수백 명에게 1초 만에 일괄 발송** |
| 3 | 1:1 담당자가 배정됩니다 | **세팅부터 정산까지, 전담 매니저가 다 해드립니다** |
| 4 | 정산 처리가 가능합니다 | **정산서 자동 발행 — 증빙 서류 걱정 끝** |
| 5 | 다양한 행사에 활용할 수 있습니다 | **축제, 설문, OT — 어떤 행사든 경품은 5분이면 준비 끝** |
| 6 | 실물 상품 대신 모바일 쿠폰으로 지급 | **포장도, 재고도, 남는 경품도 없습니다** |
| 7 | 예산에 맞게 상품을 선택할 수 있습니다 | **예산만 알려주세요. 최적 구성을 추천해드립니다** |
| 8 | 원스톱 정산이 가능합니다 | **품의서 쓰고, 영수증 모으고? 여기선 클릭 한 번입니다** |

## 5. 추가로 넣어야 할 것 — 보완 제안

### 5-1. 반드시 추가

| 항목 | 이유 | 위치 |
|------|------|------|
| **도입 대학 실적** (숫자/로고) | B2B 의사결정에서 "남들도 쓴다"는 가장 강력한 설득 | 뒷면 or 앞면 하단 |
| **실제 활용 사례 1~2건** | "OO대 축제에서 500명에게 커피 쿠폰 발송, 당일 완료" 같은 짧은 사례 | 뒷면 |
| **가격 예시표** (행사 담당자용) | 예산 감각이 바로 잡혀야 문의로 이어짐 | 뒷면 |
| **QR코드** (상담/카톡 연결) | 전단지 → 행동 전환의 핵심 장치 | 앞면 or 뒷면 CTA 영역 |

### 5-2. 강력 추천

| 항목 | 이유 |
|------|------|
| **Before/After 비주얼** | "기존 방식 5단계 vs 이음기프트 2단계" 비교 도식 — 직관적 설득력 |
| **미사용 쿠폰 환불 정책** | 교직원의 가장 큰 우려("남으면 어쩌지?")를 선제 해소 |
| **보안·개인정보 관련 한 줄** | 학교 행정은 개인정보에 민감 — "번호 암호화 처리" 등 안심 문구 |
| **시즌별 추천 활용표** | 3월 OT → 5월 축제 → 9월 설문 → 12월 복지 — 연중 활용 가능성 어필 |
| **최소 주문 수량/금액** | "10건부터 가능" 같은 진입 장벽 낮추기 |

### 5-3. 선택 추가 (차별화 요소)

| 항목 | 이유 |
|------|------|
| **학교 전용 할인/프로모션** | "대학 첫 이용 시 수수료 무료" 같은 한정 혜택 — 행동 유도 |
| **브랜드 라인업 로고** (스타벅스, CU, 배달의민족 등) | 익숙한 브랜드가 보이면 신뢰도 + 상품 기대감 상승 |
| **담당자 프로필/사진** | 1:1 전담 매니저를 강조하려면 실제 인물감 부여 |
| **후기/추천사** | "편해서 매 학기 쓰고 있습니다" — 짧은 한 줄 추천 |

## 6. 최종 체크리스트

전단지 제작 전 아래 항목을 확인하세요:

- [ ] 헤드라인이 기능이 아니라 **고민 해결**로 시작하는가?
- [ ] 3초 안에 "뭘 해주는 건지" 파악되는가?
- [ ] 교직원용과 행사 담당자용 **톤·메시지가 구분**되는가?
- [ ] **숫자/수치**가 최소 3개 이상 들어가 있는가?
- [ ] **도입 실적** 또는 **활용 사례**가 포함되어 있는가?
- [ ] **CTA(행동 유도)**가 명확하고 쉬운가? (QR, 전화번호)
- [ ] 뒷면에 **가격 감각**을 잡아주는 정보가 있는가?
- [ ] "남는 쿠폰은?" "개인정보는?" 같은 **우려 선제 해소**가 있는가?
