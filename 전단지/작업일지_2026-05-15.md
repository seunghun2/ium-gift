## 1. 상품 단가표 (v5_비공개단가표.html) 수정 및 빌드
* **기획전 상품 특수 로직 적용:** 엑셀의 기획전 시트에서 파싱한 상품 중, 할인율이 18%, 20%, 25%, 30%인 특별 상품들은 15%, 20%, 22% 등 어떤 구매 구간 필터를 누르더라도 금액과 할인율이 변동 없이 **해당 고정 할인율을 유지**하도록 로직(`build_pricing.py`) 변경.
* **단가표 ➡️ 인테이크 폼 이동 링크 수정:** 페이지 하단의 '맞춤 견적 상담받기' 버튼 클릭 시 이동하는 경로를 최신 버전인 `v6_인테이크_모바일.html`로 업데이트.
* **푸터 로고 오류 수정:** 로컬 및 라이브 서버에서 엑박으로 뜨던 푸터 데브늄 로고의 상대 경로를 올바른 경로(`./assets/logo.png`)로 재설정.

## 2. 모바일 인테이크 폼 (v6_인테이크_모바일.html) 고도화
* **폼 입력 3단계(Step) 분할:** 
  - (1/3) 신청자 정보
  - (2/3) 소속 정보
  - (3/3) 행사 및 예산 정보
  - 기존 2단계 구조를 3단계로 쪼개어 모바일 사용성을 높이고 진행률(Progress Bar) 로직 동기화 완료.
* **소속 인증 서류 선택화:** 기존 필수 제출값이던 서류 첨부 필드의 `*` 표시를 제거하고 `(선택)`으로 변경. 파일 미첨부 시에도 폼이 넘어가도록 유효성 검사 해제.
* **잔디(JANDI) 웹훅 실시간 연동:** 
  - 고객이 3단계 입력을 모두 마치고 [신청 완료하기] 버튼을 누를 시, 입력된 내용(이름, 연락처, 소속, 부서, 목적, 예산)이 **잔디 메신저 토픽방으로 즉시 전송되도록 API Fetch 스크립트 작성 및 URL 매핑 완료.**

## 4. 추가 수정 및 배포 (2차)
* **파일 네이밍 및 경로 최적화:** 
  - `v6_인테이크_모바일.html`을 `info.html`로 직관적으로 변경.
  - 관련 페이지(`build_pricing.py`)의 버튼 링크를 `info.html`로 업데이트 적용.
* **랜딩 페이지 연동 (404 에러 해결):** 
  - Vercel 배포 디렉토리에 기존 `landing` 폴더(인덱스 및 스타일 등) 누락분을 포함시킴.
  - `info.html` 내 '캠퍼스 파트너 서비스 알아보기' 버튼들이 정상적으로 랜딩 페이지(`./landing/index.html`)로 향하도록 경로 연결.
* **푸터 로고 사이즈 업그레이드:** 단가표 하단의 데브늄 로고가 너무 작다는 피드백 반영, 가로 너비를 140px로 키우고 세로 비율은 자동(`height: auto`)으로 늘어나게끔 조정하여 재배포.

## 5. 인테이크 폼 (info.html) 입력 검증 강화
* **전화번호 유효성 검사:** 
  - 모바일에서 숫자 키패드 자동 활성화 (`inputmode="tel"`)
  - `010`으로 시작하는 총 11자리 번호가 아니면 빨간색 에러 메시지를 표시하고 다음 단계로 넘어가지 않도록 차단.
  - placeholder를 `010-0000-0000`으로 변경하여 입력 형식 안내.
* **완료 페이지 수정하기 버튼 제거:** 신청 완료 후 `[수정하기]` 버튼을 삭제하고 `[이음기프트 홈으로 가기]` 버튼만 풀 너비로 정리.
* **홈 버튼 링크 변경:** 완료 페이지의 `[이음기프트 홈으로 가기]` 버튼 클릭 시 공식 도메인(`https://www.ium-gift.com/`)으로 이동하도록 수정.

## 6. 랜딩 페이지 (landing/index.html) 잔디 웹훅 연동
* **상담 신청 폼 잔디 연동:** 랜딩 페이지 하단의 '무료 비교 견적서 받기' 상담 신청 폼에서 `[상담 신청하기]` 버튼 제출 시, 대학교명/이름/직책/전화번호/예산 규모 데이터가 **잔디 메신저 토픽방으로 실시간 전송**되도록 Fetch API 스크립트 추가.
  - 잔디 메시지에 `📋 [랜딩페이지]` 태그를 붙여 기존 인테이크 폼(`🎉 [프로모션 문의]`)과 **유입 경로를 구분** 가능.
* **예산 규모 입력 필드 추가:** 랜딩 폼에 셀렉트박스(100만원 미만 / 100~300만원 / 300~500만원 / 500만원 이상 / 미정) 추가. 웹훅 데이터에도 포함.
* **배포 동기화 이슈 해결:** `_deploy/landing` 폴더에 변경사항이 반영되지 않던 문제를 발견. 기존 폴더 삭제(`rm -rf`) 후 원본에서 새로 복사하는 방식으로 수정하여 해결.

## 7. 랜딩 페이지 상담 폼 (landing/index.html) UI/UX 고도화
* **할인 혜택 강조 문구 추가:** 폼 상단에 `정가보다 저렴하게 · 대학 전용 할인가 최대 25% 할인` 배지를 추가하여 핵심 혜택을 시각적으로 강조.
* **입력 필드 확장:** 
  - **행사 내용 필드 추가:** '행사 내용' 인풋 박스를 추가하여 신입생 환영회, 축제 등 구체적인 상담 목적을 수집 가능하도록 개선.
  - **웹훅 연동:** 추가된 '행사 내용' 데이터 역시 잔디 메신저 알림에 실시간으로 포함되도록 로직 업데이트.
* **디테일 개선:** 
  - 예산 규모 선택(Select) 박스 내부에 화살표 아이콘(`chevron-down`)을 추가하여 드롭다운임을 명확히 인지 가능하도록 UI 개선.
  - 불필요한 클립보드 아이콘을 제거하여 폼 상단을 깔끔하게 정리.
* **문구 일관성 확보:** 서비스 소개 섹션과 상담 폼 상단 배지의 문구를 `정가보다 저렴하게 · 대학 전용 할인가 최대 25% 할인`으로 통일하여 사용자에게 일관된 메시지 전달.
