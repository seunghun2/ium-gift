## 원본 → 분할 매핑

```
┌─────────────── 원본 (단면) ──────────────────┐
│                                               │
│  🔵 히어로 (헤드라인 + 폰 목업)                 │ ──→ 앞면 상단
│  🔵 이음기프트가 필요한 이유 (8가지)             │ ──→ 앞면 하단
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │  ← 여기서 쪼갬
│  🟠 실제 대학 활용 시나리오 (7가지)              │ ──→ 뒷면 상단
│  🟠 추천 패키지 제안 (4종)                      │ ──→ 뒷면 중단
│  🟠 도입 혜택 아이디어 (5가지)                   │ ──→ 뒷면 하단
│  🟠 배너 카피 + CTA                            │ ──→ 뒷면 하단
│                                               │
└───────────────────────────────────────────────┘
```

## 앞면 구성 (WHY — 왜 써야 하는지)

### Zone A · 히어로 (상단 40%)

| 요소 | 내용 |
|------|------|
| 상단 태그 | 학생회·교내기관을 위한 모바일 쿠폰 지급 솔루션 |
| 헤드라인 | **학교 행사·복지·보상 운영, 이제 더 간편하게!** |
| 서브카피 | 지급부터 발송·관리·정산까지 한 번에, 이음기프트 |
| 비주얼 | 폰 목업 (모바일 쿠폰 도착 화면) |
| 4대 키워드 | 모바일 쿠폰 기반 / 예산 맞춤 / 대량 발송 / 정산 원스톱 |

### Zone B · 필요한 이유 8가지 (하단 60%)

| # | 키워드 | 설명 |
|---|--------|------|
| 01 | 누구나 쉽게 상품 지급 | 학생회·행정팀·학과 모두 쉽게 사용 |
| 02 | 운영 부담 최소화 | 실물 준비·보관·배포 부담 없음 |
| 03 | 다양한 활용처 | 행사, 설문, 이벤트, 복지 등 폭넓게 |
| 04 | 예산 맞춤 상품 | 원하는 예산과 규모에 맞춰 유연하게 |
| 05 | 대량 발송·관리 | 많은 인원도 한 번에 |
| 06 | 1:1 담당자 배치 | 전담 담당자가 배정되어 편리한 커뮤니케이션 |
| 07 | 정산까지 원스톱 | 발송부터 정산까지 한 번에 처리 |
| 08 | 시간·인력 절감 | 실물 운영 대비 업무 효율 대폭 향상 |

## 뒷면 구성 (HOW — 어떻게 쓰는지)

### Zone C · 활용 시나리오 (상단 35%)

| 시나리오 | 아이콘 | 설명 |
|---------|--------|------|
| 학생회 행사 경품 | 선물 | 학생회 주관 행사 참여자 대상 경품을 간편하게 모바일로 지급 |
| 축제/이벤트 리워드 | 텐트 | 축제 스탬프 투어, 부스 참여 이벤트 등 프로모션에 즉시 활용 |
| 설문/만족도 조사 보상 | 클립보드 | 설문·만족도 조사 참여자에게 즉각 보상 |
| 신입생/재학생 웰컴 리워드 | 환영 | 신입생 환영 오리엔테이션, 학과 보상 |
| 교직원 복지/감사성 지급 | 하트 | 교직원·조교·강사 대상 복지성 지급이나 감사 |
| 비교과 프로그램 참여 보상 | 트로피 | 학교 프로그램, 특강, 동아리 참여 대상 보상 |
| 교내 캠페인/홍보 이벤트 | 메가폰 | 교내 캠페인, SNS 이벤트 참여자 리워드로 |

### Zone D · 추천 패키지 (중단 30%)

| 패키지 | 대상 | 주요 내용 |
|--------|------|----------|
| **기본형** | 소규모 행사·상시 지급용 | 소규모 이벤트, 상시 보상 지급 · 간편한 발송과 관리 · 운영 부담 최소화 |
| **행사형** | 축제·학생회 이벤트·대형 행사 운영 | 대량 발송이 필요한 행사에 최적 · 다양한 상품 구성 가능 · 실시간 발송/관리 |
| **설문형** | 만족도 조사·참여 보상 지급용 | 설문, 만족도 조사 보상에 특화 · 참여율 향상에 효과적 · 자동 발송으로 편리한 운영 |
| **복지형** | 교직원·조교·내부 구성원 복지용 | 교직원 복지, 감사성 지급에 적합 · 예산 계획에 맞춘 안정적 운영 · 정산까지 간편한 원스톱 관리 |

### Zone E · 도입 혜택 + CTA (하단 35%)

| 혜택 | 설명 |
|------|------|
| 첫 이용 학교 초기 운영 지원 | 새롭게 도입하는 학교를 위한 초기 운영 지원 |
| 행사 시즌 한정 추가 혜택 | 축제·행사 시즌에 맞춰 추가 혜택 제공 |
| 누적 발송 구간별 추가 혜택 | 쿠폰 발송 규모에 따른 추가 혜택 제공 |
| 학기 초 집중 운영 프로모션 | 학기 초 추가 시기에 맞춘 집중 운영 프로모션 |
| 대량 발송 이용 시 운영 조건 우대 | 대량 발송 이용 시 추가 운영 조건 우대 |

**CTA:**
- 지금 상담 신청하고 우리 학교 맞춤 운영을 시작하세요!
- 상담 신청 → 맞춤 제안 → 간편 운영 시작
