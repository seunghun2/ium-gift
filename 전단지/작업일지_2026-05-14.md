## ✅ 완료 항목

### 1. 데이터 전체 교체
- 새 엑셀 `(1)` 파일로 전체 금액 재추출 (상품권 50건 / 일반상품 209건)
- 할인율 소수점 제거 → **정수 표기** (e.g. `19.8%` → `20%`)
- 절삭가 기준으로 정확한 할인율 재계산

### 2. 혜택 카드 아이콘 + 태그 교체
- 이모지(💰🎁💵) → **Lucide 아이콘** (`percent`, `gift`, `wallet`) 16px 컬러 매칭
- 태그 워딩 변경:
  - `추천` → **가성비 BEST**
  - `활용도 만점` → **복지 활용**
  - `법인 추천` → **정산 간편**
- VIP 이모지(🛡️) → `shield-check` 아이콘

### 3. 예산 파악하기 섹션
- 문구 변경: `예산 시뮬레이터` → **예산 파악하기**
- 3개 옵션 탭 추가: `즉시 할인` / `상품권 페이백` / `현금 페이백`
- 각 탭별 계산 로직 분리:
  - 즉시 할인: 15%/20%/22% 자동 적용 → 절감 금액
  - 상품권/현금 페이백: 기본 15% + 추가 5~7% 페이백 금액

### 4. 가격 티어 셀렉터 UX 개선
- 라벨 간결화: `100만원 이상 구매시 20%` → 2줄 구성 (`100만원 이상` / `20%`)
- 버튼 높이 확보 + 가독성 향상

### 5. 필터 영역 UX 2차 개선
- **가격대 필터 칩**: 활성화 시 검정색 폰트 + 옅은 회색(`slate-200`톤) 배경으로 변경하여 시각적 피로도 감소
- **라벨 제거**: 불필요하게 공간을 차지하던 `🏷 카테고리`, `💴 가격대` 텍스트 라벨 삭제 (심플한 UI 구성)

### 6. 예산 기반 동적 기능 고도화 (Magic Sync)
- **예산-티어 완벽 동기화**: 
  - 사용자가 예산을 입력하면 그에 맞는 할인 티어(15%/20%/22%) 버튼이 자동으로 활성화
  - 반대로 티어 버튼 클릭 시 해당 티어의 최소 금액(예: 100만 원, 1,000만 원)이 예산 입력창에 자동 채워짐
- **상품 리스트에 예상 구매 내역 표시**: 예산 입력 시 하단 단가표의 개별 상품마다 "해당 예산으로 구매 가능한 총 개수"와 "총액" 실시간 노출 (예산 부족 시 "예산 부족" 경고)
