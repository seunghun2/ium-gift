## 2. 세부 작업 내용 (Changelog)

### A. 메인 프로모션 랜딩페이지 (`landing/index.html`)
1. **히어로(Hero) 섹션 카피라이팅 최적화**
   * 기존: "카톡으로 한 명씩 50번 보내고 계세요?"
   * **수정:** "아직도 카카오톡으로 정가에 구매하고 일일이 선물하고 계신가요?" (고충 타격 강화 및 어색한 줄바꿈 `word-break: keep-all;`로 해결)
2. **비즈니스 용어 순화**
   * "B2B 특가" ➡️ "대학 전용 할인가"
   * "후불 결제" (기능 삭제) ➡️ **"완벽한 행정 증빙"** (법인카드, 세금계산서 발급 강조로 대체)
3. **서비스 플로우(4단계) 현실화**
   * 무료 체험 쿠폰 지급 중심에서 **'무료 맞춤 컨설팅 및 1장짜리 비교 견적서 제공'** 흐름으로 섹션 전면 교체.
4. **앱 레이아웃 적용**
   * 인테이크 폼과 동일하게 최대 가로폭 480px의 `.mobile-container`를 씌우고, 상단 좌측에 뒤로가기 화살표를 배치해 페이지 간의 자연스러운 이동(퍼널) 구축.

### B. QR코드 전용 견적 신청 폼 (`v5_인테이크_모바일.html`)
1. **상단 및 메인 UI 개선**
   * 상단 이음기프트 로고 사이즈 확대 (100px ➡️ 140px).
   * 뒤로가기 화살표가 제목과 겹치는 현상 해결 (`position: absolute` 속성 제거 및 여백 조정).
2. **입력 폼(Input) 유효성 및 편의성 강화**
   * **소속 라디오 버튼:** 선택 시 부담스러운 솔리드 보라색 대신, 은은한 연보라색 배경(`#EFECFF`)에 보라색 폰트로 세련되게 변경.
   * **셀렉트 박스:** 브라우저 호환성을 타는 CSS 가짜 화살표 대신, HTML에 직접 `Lucide` 아이콘(chevron-down)을 박아 넣어 명확한 인지 가능.
   * **휴대폰 번호 방어막:** `maxlength="11"` 제한, `inputmode="numeric"` 키패드 호출, 자바스크립트 정규식을 통해 한글 및 특수문자 타이핑 원천 차단.
   * **대학 맞춤 라벨링:** "팀장/회장" 등 딱딱한 직책을 "교직원, 기획국장, 조교, 총무 등" 대학교 실무진 언어로 교체.
3. **개인정보 동의 영역 아코디언 토글화**
   * 지저분하게 나열된 약관 텍스트를 숨기고, 체크박스와 우측 화살표(`v`)를 분리.
   * 체크박스/글씨 클릭 시엔 체크만 되고, 화살표 클릭 시에만 상세 내용이 펼쳐지도록 클릭 영역 완벽 분리 및 수직 중앙 정렬 달성.
4. **임시저장 기능 고도화**
   * 글자를 칠 때마다 화면을 가리던 거슬리는 토스트 팝업 제거.
   * 백그라운드에서 사용자 모르게 30초 간격으로 자동 저장(`setInterval`)되도록 변경.
   * 하단 CTA에 명시적인 `[임시저장]` 버튼(회색 배경)을 신설하여 안도감 제공.
5. **투박한 Alert 팝업 ➡️ Step 3 완료 화면으로 전환**
   * 폼 제출 시 상단에 뜨던 투박한 브라우저 알림창(`alert`)을 삭제.
   * 제출 완료 시 부드럽게 기존 화면이 사라지고, 체크 모양 아이콘과 함께 "신청이 완료되었습니다"라는 Step 3 완료 화면이 렌더링되도록 구조 개편.
   * 완료 화면에 작성 정보 요약(영수증형 레이아웃) + 수정하기 버튼 추가.
6. **문구 일괄 순화 (토스 톤앤매너)**
   * "솔루션" → "서비스", "혜택 적용" → "맞춤 견적 안내", "인증 서류" → "확인용 서류" 등 8곳 교체.
   * 서브타이틀: "부담 없이 적어주세요, 빠르게 연락드릴게요."
7. **행사/목적 복수 선택 체크박스 전환**
   * 텍스트 입력 → 6개 체크박스(신입생 환영, 시험기간 간식, 축제/체육대회, 교직원 포상, MT/워크숍, 기타) + 기타 텍스트 입력.
8. **소속 확인용 서류 첨부 선택 전환**
   * 필수(*) → 선택으로 변경하여 진입 장벽 낮춤.
