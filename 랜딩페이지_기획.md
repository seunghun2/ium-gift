# 이음기프트 캠퍼스 파트너 — 프로모션 랜딩페이지 기획

## 전환 퍼널

```
전단지 QR 스캔
    ↓
프로모션 랜딩페이지 (이 페이지)
    ↓
등록 폼 작성 (대학명, 이름, 직책, 전화번호)
    ↓
"체험 쿠폰이 발송됩니다" 안내
    ↓
이음기프트 팀에 알림 → 쿠폰 직접 발송 + 영업 연결
```

## 기술 노트

- **폼 제출 처리**: 제출 시 이음기프트 팀에게 알림 (이메일/슬랙/카톡)
- **쿠폰 발송**: 자동이 아니라 팀이 직접 발송 (수동 대응)
- **호스팅**: 대학 전용 서브도메인 or 별도 URL 권장
  - 예: campus.ium-gift.com 또는 ium-gift.com/campus
- **모바일 최적화 필수**: 전단지 QR → 스마트폰으로 접속할 확률 높음
- **GA 추적**: UTM 파라미터로 어느 대학 전단지에서 유입됐는지 추적
